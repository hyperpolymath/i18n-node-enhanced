# Prohibited Uses
# ===============
prohibited-uses: malware-generation, vulnerability-exploitation, license-circumvention
prohibited-uses-detail: |
  - Do not use to generate malicious internationalization payloads
  - Do not use to circumvent security features
  - Do not use to create competing proprietary forks without proper attribution
  - Do not use training data to infer or expose sensitive enterprise configurations

# Ethical Guidelines
# ===================
ethical-use: required

  - Respect user privacy in translation data
  - Maintain accessibility standards
  - Support linguistic diversity
  - Avoid cultural bias in automated translations
  - Preserve security best practices
